
Security Titanium

****Descripción****
Sistema de gestión para una empresa de asesoria en prevención de riesgo.
Este software de gestión ofrece las herramientas necesarias para llevar registro de todas las actividades
de la empresa de forma eficiente y segura, todo en un mismo sistema.También permite acceso diferido a la plataforma
a los distintos usuarios.
